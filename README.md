**3D EXPERIENCE Web Services example with C# REST API**
  - Clone the project.
  - Go to App Environment folder.
  - Check application paths. 
  - Check 3DEXPERIENCE urls and write your cloud tenants.
  - Check NuGet packages.
  - Click Run & Test
